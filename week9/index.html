Notes

            Browser Object Model or BOM is a collection of properties and has info about the browser and computer screens.
            There is no official standard of the BOM.

            There are dialog boxes like Alert and confirm and prompt, but there is also those for window. Like window.alert().

            In the Window object there is a navigator property to have information about which browser the user is currently using. This is kind of misleading since users can masquerade as other browsers.

            There are many different properties that give information about different aspects of browsers going from the history to locations of the window.

            window.open will allow you to set a url and size of a window of a browser to show a second page when needed.
            You can also have popup.close() function to close previous opened small windows.

            Can set cookies by document.cookie = 'name=Superman';
            To delete a cookie you go 
                document.cookie = 'name=Batman; expires=Thu, 01 Jan 1970 00:00:01 GMT';
            You can have popups have a timeout by going window.setTimeout(()=> alert("Time's Up!"), 3000);

            the Data Attribute is a way of putting data into custom attributes that the browser ignore. They are also private to a page.

            Geolocation will get the geographical position of the device. This is able to give user location pretty exactly.
            Can use 
                navigator.geolocation.getCurrentPosition(youAreHere);

            Websockets is a new protocol that allows two way communication with a server.

            YOu can add audio files through audio tags and you can add video by video tags.
            Videos you can put volume and pause functions and attributes to edit the video on a page.
