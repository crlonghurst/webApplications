Notes

          There are transforming functions in CSS3, it is called by using transform and using another name separated by a space.
          Such as:
            transform: translate(45px, -45px);
          Translation allows for moving elements left, right, up or down. Similar to the behavior of position: relative; when declaring top or left.
          Unlike position: relative translate only moves based on its current position.

          Scale scales items by defined parameters horizontally then vertically.
           transform: scale(1.5, 0.25);
           Can use scaleX(x) or scaleY(x) these will only scale the present direction of the function.

          Rotate will rotate the element on a radius.

          You can set the origion of the transform by:
            transform-origin: 0 0;
          
            To support older Browsers you need to include -webkit- and put it on every transform as a prefix. 
            Like:
                -webkit-transition-property: -webkit-transform;
                transition-property: transform;
            There is a Transition-duration that sets how long the transition will last and how long it will take to get to the transition.

            transition properties allow for multiple transitions in one call.
            Such as:
                transition-property: transform, color;
                transition-duration: 0.2s;
                transition-timing-function: ease-out;
                transition-delay: 50ms;
            
            There is a keyframes that allow for animations have to use @-webkit-keyframes myAnimation{
                code
            }
            Animation properties:
                animation-name:
                animation-duration:
                animation-timing-function
                animation-iteration-count:
                animation-direction:
                animation-direction:
                animation-delay:
                animation-fill-mode:
            
            Warning with naming the animation and using multiple animations in one call such as:
                animation:
                    animationOne 300ms ease-in backwards,
                    animationTwo 600ms ease-out 1s forwards;
